# Static Website Hosting using Amazon S3 🌐

## 📌 Project Overview

This project demonstrates how to host a static website using Amazon S3 without using any server.

## 🧰 AWS Services Used

* Amazon S3

## ⚙️ How It Works

A static HTML website is uploaded to an S3 bucket.
S3 static hosting is enabled, and the website is made publicly accessible.

## 🧪 Steps

1. Create S3 bucket
2. Disable block public access
3. Enable static website hosting
4. Upload index.html
5. Add bucket policy
6. Access website using S3 URL
